3Cheap as hell

greendot is scripted in PHP. PHP is supported by more or less every webspace arangement and is a webwide standard. PHP web hostings are cheapt to get (there are also free offers).
PHP and the database MySQL are open source and free available. Also the most extensions for this two technologies.

4Full controll

The framework itself is very small and only comes with a "page delivery" module and some core functions like multilingual support and so on. But it also provides uncountable other modules which you can put on top of the framework and extend it's funcitonality if needed. (e.x. database support, form valdation, facebook API, etc.) So you have always the full controll over the components and the ability of your framework installation.
